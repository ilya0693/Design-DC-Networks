###Конфигурация VPCS

Первым шагом сконфигурируем устройства клиентов, которые, в дальнейшем, будут взаимодействовать между собой в рамках VxLAN фабрики.

Конфигурация VPCS **_"Client1"_**

```sh
set pcname Client1
ip 10.123.100.10 255.255.255.0 10.123.100.1
```
